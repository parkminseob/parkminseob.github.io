* 배열 뒤집기(암기)
  절반으로 나눠서 알고리즘을 푼다.

```
static Object[] reverse(Object[] arr) { 
 for(int i = 0; i < arr.length / 2; i++){
	Object temp = arr[i];
	int targetIndex = arr.length - 1 -i;
	arr[i] = arr[targetIndex];
	arr[targetIndex] = temp;
  }
return arr;
}
```
